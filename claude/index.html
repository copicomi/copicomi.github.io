CLAUDE.md #

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

项目概述 #

这是一个基于 Hugo 静态网站生成器的个人技术博客，使用 Typo 主题，内容主要通过 Obsidian 管理和编辑。

常用命令 #

本地开发 # 

# 从项目根目录运行
hugo server -D          # 启动开发服务器（包含草稿）
hugo server             # 启动开发服务器（不含草稿）

构建与部署 # 

hugo                    # 构建静态网站到 public/ 目录
./push.sh               # 部署到 GitHub Pages（会强制推送）

内容结构 #

文章组织 #

  • 位置: posts/ 目录
  • 格式: 每篇文章是一个子目录，包含 index.md 文件和可选的 assets/ 图片目录
  • Frontmatter: 使用 TOML 格式（+++ 包围） 
    +++
title = "文章标题"
date = "2025-10-26"
summary = "文章摘要"
categories = ["分类"]
tags = ["tag1", "tag2"]
draft = false    # 草稿设为 true
math = true      # 需要数学公式时启用
+++

主要内容系列 #

  • 重读 xv6: 操作系统学习笔记，分为 I-VII 七个部分
  • pager-reading: 论文阅读笔记
    • S081/: MIT 6.S081 相关论文
    • 分布式/: 分布式系统论文（GFS, Bigtable, Chubby, Paxos, Spanner）
  • 课程复盘: 数据库课程实验总结
    • 15445 复盘: CMU 15-445 Database Systems
    • TinyKV 复盘: PingCAP 的分布式 KV 课程
  • 实现系列: 从零实现的数据结构和算法
    • 从头写一个 B+Tree
    • 写 Raft 时获得的一些方法论
  • 源码阅读:
    • Leveldb 源码笔记

图片资源 #

  • 放置在对应文章目录的 assets/ 子目录中
  • 在 Markdown 中引用: !["alt text"](assets/filename.png "title")

主题与配置 #

Typo 主题 #

  • 主题位置: themes/typo/
  • 详细文档: https://tomfran.github.io/typo-wiki/
  • 特性: 暗色模式、自动标题编号、多种配色方案、评论支持

配置文件 #

  • 主配置: hugo.toml（项目根目录）
  • 自定义布局: layouts/partials/head.html（覆盖主题的 head）
  • 文章模板: archetypes/default.md

税法（Taxonomies） #

  • tags: 文章标签
  • categories: 文章分类

内容管理 #

Obsidian 集成 #

  • .obsidian/ 目录包含 Obsidian 的配置和插件
  • 使用 Obsidian 进行内容编辑和管理
  • 注意 Obsidian 的同步冲突文件（.sync-conflict-*）应被忽略

文章创建 #

使用 Hugo archetype 创建新文章：

hugo new posts/新文章/index.md

注意事项 #

  1. 路径问题: 主题文件和配置文件在项目根目录（../），当前工作目录是 content/
  2. 中文内容: 博客主要为中文技术文章，保持中文写作风格
  3. 图片路径: 相对于文章目录，使用 assets/ 子目录
  4. 数学公式: 需要在 frontmatter 中设置 math = true
  5. 部署: push.sh 使用强制推送，确保在正确的分支上操作
  6. 草稿: 设置 draft = true 时，需要 hugo server -D 才能预览

技术栈 #

  • 静态生成器: Hugo
  • 主题: Typo (https://github.com/tomfran/typo)
  • 编辑器: Obsidian
  • 部署: GitHub Pages
  • 版本控制: Git
